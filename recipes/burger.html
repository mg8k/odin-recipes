Burger Recipe

Description

This is a recipe for a delicious burger that will be classic but delicious.

it will include some onions tomatoes and a delicious ranch sauce with some cheese

Ingredients

  • Beef Patty
  • Bun
  • Tomato
  • Onion
  • Ranch Sauce

Steps

  1. Cook the beef patty with some oil
  2. add some butter when almost done
  3. cut up onion and tomato into slice
  4. add the patty and the veggies into a bun
